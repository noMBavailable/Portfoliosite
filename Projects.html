Projects

Horse Race

We are developing a game for the "Fries landbouw museum" in Leeuwarden. In the game you play as someone riding a horse traversing the land while answering questions based on the museum. The movement is based on templerun, with having 3 lanes to horizontally move around in. I'm currently leading this project with another fellow student and we have the responsibility of guiding the team in the right direction while dealing with documentation and planning. We have 4 developers with us while making this project, so in total 6 people and are using scrum. For the project we are making a wooden horse to sit on, thats the 3d sketchup image as the last slide. We are also making vr controllers disguised as reins to control the horse. This game is made in unity with c# as the scripting language.

Ascending Madness

This game is an unfinished dark souls like game where you ascend levels while fighting enemies. The concept is that per level, waves of enemies increase, and you have to kill them all to go the the next level. The coontrols are based on dark souls, with the signature roll, a triple attack, blocking, and future parying. This game is made in unity with c# as the scripting language.

VR-Bowling

This is a VR-Bowling game with a working scoreboard and an official score counter. The game is played with vr controllers, where you grab the ball, aim, and throw it towards the pins. The scoreboard keeps track of your score and displays it after each frame. The game is made in unity with c# as the scripting language.

For the cause

A pixelart 2D platformer with multiple levels and powerups, with the goal the help clean the world. You play as a character who collects trash and recycles it while avoiding enemies and obstacles. The game features 3 levels with increasing difficulty and powerups like slowtime. The game is made in unity with c# as the scripting language.
